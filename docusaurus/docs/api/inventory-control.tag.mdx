Start and stop RFID tag reading (inventory) operations on the scanner.

| Command | What It Does |
|---|---|
| `control_operation` | Start or stop an RFID inventory scan |

While running, the scanner continuously reads RFID tags and publishes tag data events to the MQTT broker.
