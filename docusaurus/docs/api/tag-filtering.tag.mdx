Control which tags are reported after an inventory scan by applying post-read filters.

| Command | What It Does |
|---|---|
| `set_post_filter` | Define filter rules "” match by EPC pattern, memory bank, or mask to include or exclude specific tags |
| `get_post_filter` | Retrieve the current tag filter configuration |

Post-filters reduce noise by only reporting tags that match your criteria.
